03

Position

⚠ position: absolute posiciona no pai mais próximo com position ≠ static. Sem esse pai → vai para o <body>.
static — padrão

Fluxo normal. top/left NÃO funcionam.

STATIC
preview
.box { position: static; } /* default */
relative

Ocupa espaço original. top/left empurram visualmente. Cria contexto para absolute.

RELATIVE
top:10 left:10
preview
.box { position: relative; top: 10px; left: 10px; }
absolute — fora do fluxo

Sai do fluxo. Posiciona no pai com position≠static.

normal
ABS
top:8 right:8
preview
.pai { position: relative; } /* obrigatório */ .filho { position: absolute; top: 10px; right: 10px; }
fixed — fixo na viewport

Não rola com a página. Relativo à janela do browser.

header fixo → position:fixed; top:0; width:100%;
.header-fixo { position: fixed; top: 0; left: 0; width: 100%; z-index: 999; }
sticky — gruda no scroll

relative até atingir o top, depois fica preso.

STICKY — top: 0
conteúdo 1
conteúdo 2
conteúdo 3
conteúdo 4
preview
.nav { position: sticky; top: 0; z-index: 100; } /* pai não pode ter overflow:hidden */
z-index — camadas

Só funciona com position ≠ static. Maior = na frente.

z-index:1
z-index:2
preview
.overlay { position: absolute; z-index: 10; }
💡 Centralizar absolute no pai → top:50%; left:50%; transform:translate(-50%,-50%);
04

Display & Layout

display: block

Ocupa linha inteira. Aceita width/height/margin/padding.

bloco 1 — ocupa linha toda
bloco 2 — nova linha
preview
/* Nativos block: div, p, h1-h6, section, header, footer */ .el { display: block; }
display: inline

Fica na linha. NÃO aceita width/height.

Texto inline continua aqui na frase. preview
/* Nativos inline: span, a, strong */ .el { display: inline; }
display: inline-block

Fica na linha E aceita width/height. Ideal para botões.

box 1 box 2 box 3 preview
.btn { display: inline-block; width: 80px; padding: 8px; }
none vs hidden + centralização
CENTRALIZADO
preview
.hidden { display: none; } /* sai do layout */ .invisible { visibility: hidden; } /* ocupa espaço */ /* centralizar com flex */ .pai { display: flex; justify-content: center; align-items: center; height: 200px; } /* centralizar só horizontal */ .box { max-width: 400px; margin: 0 auto; }

06

Box Model & Espaçamento

margin — espaço externo
border
padding — espaço interno
CONTENT
width × height

↑ Todo elemento tem essas 4 camadas.

// espaçamento
/* MARGIN — espaço FORA */ margin: 20px; margin: 10px 20px; /* top/btm lft/rgt */ margin: 10px 20px 15px 5px; /* T R B L */ margin: 0 auto; /* centraliza */ /* PADDING — espaço DENTRO */ padding: 20px; padding: 16px 32px; padding-top: 40px; /* BOX-SIZING — SEMPRE USAR */ * { box-sizing: border-box; } /* padding e border ficam DENTRO do width */ overflow: hidden | auto | scroll; min-height: 400px; height: 100vh; /* 100% da janela */
⚠ Sem box-sizing:border-box, padding faz o elemento CRESCER além do width definido.
